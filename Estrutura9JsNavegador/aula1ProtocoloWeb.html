URLs

cada arquivo e carregado no navegador tem uma URL

a Unifor Resouce Locator pode ser dividida em 3 partes

por exemplo: https: //horadecodar.com.br/ index.html

https e o PROTOCOLO - horadecodar.com.br -- e o dominio,que referencia um servidor (DNS > IP)

e index.html - o arquivo/pagina que estamos acessando


HTML

HyperText Markup Languenge, e uma linguagem de marcação, onde estruturamos as paginas web, criando elementos. Os elementos sao chamados de TAGS que podem ser titulo,imagens,formularios,listas...etc

as TAGS sao caacterizados por

texto

-- e tambem podemos adicionar o CSS


toda pagina html tem 2 partes inportante - head - body . No head inserimos as configurações da pagina e importaçoes de outros arquivos - css - js-

ja no body - temos os elementos que ficam visiveis para o usuario. As TAGS possuem ATRIBUTOS que coniguram elementos


HTML E JS


podemos adicionar js ao html por meio da tagscript em arquivo externo ou script na pagina. Algumas TAGS tem ATRIBUTOS que podemos executar js, mas isso nao e utilizado

sempre que houver um link entre um arquivo e outro uma chamada HTTP e executada. js pode ser utilizado para MANIPULAR elementosdo html e alterar estilos
