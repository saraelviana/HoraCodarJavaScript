Assincrona

a programação assincrona precisa ser utilizada quando as respostas nao sao obtidas de forma imediata no programa

chamadas a uma API sao assincronas,nao sabemos quanto tempo a resposta pode demorar

ate agora utilizamos so instrumentaçoes sicronas

na programação assincrona as execuççoes nao ocorrem em formato de fila,e sim no seu tempo

assincrona para o usuario seria o LOAD -- pois nao sabem quando vira a resposta

se dizer que SINCRONA "trava" o programa, pois ele para ate terminar o processo do programa

no assicrona o programa anda mesmo lento, esta e uma vantagem
